# Tic Tac Toe Game

A console-based Tic Tac Toe game developed in C++ using loops, arrays, functions, and conditional statements.

## Features

- Two Player Game
- Dynamic Board Display
- Win Detection
- Draw Detection
- Replay Option
- Input Validation

## Technologies Used

- C++
- Arrays
- Functions
- Loops
- Conditional Statements

## How to Run

Compile

```bash
g++ tictactoe.cpp -o game
```

Run

```bash
./game
```
